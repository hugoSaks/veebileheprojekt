# Shortcut Codex
Veebilehe projekt nimega Shortcut Codex Sissejuhatus erialasse aine raames

Shortcut Codex on kiire viiteveeb klaviatuuri kiirklahvide jaoks, mis on loodud spetsiaalselt selleks, et aidata IT- ja informaatikatudengeid kiiremini koodi kirjutada, tõhusamalt siluda ja oma õppevahendeid meisterlikult kasutada.

### Sihtrühm
See tööriist on loodud üliõpilastele ja kõigile, kes õpivad programmeerimist, tarkvaraarendust või IT-d ja soovivad oma töövoogu kiirendada.

### Funktsioonid
1. OS-i põhised kiirklahvid: Leia koheselt kiirklahvid oma keskkonna jaoks, olgu selleks MacOS, Windows või Linux.
2. Lemmikud: Salvesta kõige kasulikumad kiirklahvid (näiteks silumiskäsklused või koodi refaktoreerimise otseteed), et sa ei peaks neid uuesti otsima.
3. Kiire otsing: Leia täpselt vajalik käsklus ilma oma koodikirjutamise voogu rikkumata.
3. Lihtne kasutajaliides: Puhas ja häirevaba disain, mis annab sulle vajaliku info kätte ja laseb sul koodi juurde tagasi pöörduda.
